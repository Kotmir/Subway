# Subway

## О проекте

Subway — учебный проект, разработанный студентами Казанского технологического колледжа КНИТУ в рамках проектной работы по дисциплине «Информационные технологии».

Проект представляет собой веб-приложение, созданное для демонстрации навыков современной веб-разработки, проектирования интерфейсов и организации клиентской части приложения.

## Технологии

В проекте используются следующие технологии:

- Next.js
- React
- TypeScript
- Tailwind CSS
- PostCSS
- Node.js

## Структура проекта

```text
app/            — страницы и маршруты приложения
components/     — переиспользуемые компоненты
hooks/          — пользовательские React-хуки
lib/            — вспомогательные функции и модули
public/         — статические файлы
styles/         — стили приложения
```

## Установка и запуск

### Клонирование репозитория

```bash
git clone <repository-url>
```

### Установка зависимостей

```bash
npm install
```

или

```bash
pnpm install
```

### Запуск проекта

```bash
npm run dev
```

или

```bash
pnpm dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Цель проекта

Проект создан в образовательных целях для:

- изучения современных веб-технологий;
- освоения фреймворка Next.js;
- получения практических навыков работы с TypeScript;
- изучения принципов компонентной архитектуры;
- приобретения опыта командной разработки.
